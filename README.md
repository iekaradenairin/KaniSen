# KaniSen 🦀

Windows のデスクトップに「かにせん」が降ってくる、放置系アクセサリ。
作業中のウィンドウの上に積もり、クリックすると「さくっ」と食べられます。

## ダウンロード

[Releases](https://github.com/iekaradenairin/KaniSen/releases) からzipファイルをダウンロードし、好きな場所に置いてダブルクリック。
※インストール不要・.NET ランタイム不要

> 初回起動時に「WindowsによってPCが保護されました」と出たら、「詳細情報」→「実行」で OK。

## 操作

- **左クリック** — かにせんを食べる
- **トレイアイコン右クリック** — メニュー（図鑑 / 設定 / 終了など）

## 設定

トレイから「設定...」で開けます。
- 食べる音の声（標準 / おねえさん / かわいい / 子供 / イケボ / ダンディ）
- 音量 + 試聴ボタン
- Windows 起動時の自動起動

## アンインストール

1. exe を削除
2. `%LOCALAPPDATA%\KaniSen\` フォルダを削除
